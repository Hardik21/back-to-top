# Javascript back-to-top

Javascript back-to-top is a simple function for the website.

## Installation

```bash
npm install back-to-top
```

## Usage

```python
const jsNotif = require('back-to-top');

# pass arguments
jsNotif.message({ displayTime: 5000 });
```
